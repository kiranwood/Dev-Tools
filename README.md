# Dev Tool

**PG29 Pipelines**

## About the Project

This project is to design and implement a dev tool that improves efficiency and workflow.

### What I did:

I created a prefab spawner that will project/show the object in the scene view. When you move your mouse the projected gameobject will follow. Users can customize/randomize the scale and rotation of the gameobject and place them in the scene without touching the individual game objects. They can also alter an offset to change the position of the gameobject from the mouse location. The projection will automatically change as the user alters values.

### Controls:

`Left click`
 - Places an object in the scene view
 - A new projection will immediately show at your mouse position

`Esc`
 - Toggles on/off prefab editing
 - Removes/shows projection

## How To Open the Unity Project

**Step 1** Clone the repo and open Unity

**Step 2** Add project from disk and find the repo project. Click `Dev tools` folder inside repo 

**Step 3** Open project. Go to top left tab -> Tools -> Prefab spawner
